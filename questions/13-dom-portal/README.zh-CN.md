`Vue.js`提供了一个内置组件,将其插槽内容渲染到另一个DOM,成为该DOM的一部分。

你知道它是什么吗 ? 让我们试试👇:

```ts
<script setup>

const msg = "Hello World"

</script>

<template>
  <!-- 将以下元素渲染成`body`的子元素 -->
  <span>{{ msg }}</span>
</template>


```
